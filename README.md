## Implementation based on making the dish categories slidable

The requirement was to make the dish categories basically the titles slidable . Instead i implemented Swiper js which is a library that helps in implementing slidable contents.The respective data of each dish categories is made to a single slide

* This solution utilizes React Hooks (useRef) and Dom manipulation
* when each content is slided , It's respective category will on focus ( text color red and underline )
* On clicking dish category , it automatically slides to its respective data slide
